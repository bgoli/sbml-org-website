616 * In SBML Level&nbsp;3, an explicit data type called <code>SIdRef</code> was
617 * introduced for attribute values that refer to <code>SId</code> values; in
618 * previous Levels of SBML, this data type did not exist and attributes were

620 * data type was the same as <code>SIdRef</code> in Level&nbsp;3.  These and

629 * descend into child elements.

631 * @param oldid the old identifier.
632 * @param newid the new identifier.
633  */ public
