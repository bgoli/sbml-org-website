SpeciesTypeComponentMapInProduct_t Class Reference
Multistate, Multicomponent and Multicompartment Species_t

Detailed Description

multi Distinguishes between components in reactants versus products.

The SpeciesTypeComponentMapInProduct_t object is a child of a SpeciesReference_t (via the MultiSpeciesReferencePlugin_t) in a Reaction_t, and defines the mapping between a component in a reactant and a component in a product. The identifications of a component and the SpeciesReference_t should be sufficient to identify the component in the context of a reaction. The attributes "reactant" and "reactantComponent" can identify the component in a reactant, and the "productComponent" attribute and the product storing the mapping information can identify the component in a product.
